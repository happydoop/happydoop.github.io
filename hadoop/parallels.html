¶ 开始安装↓

此时鼠标会被虚拟机捕获，不能在 Mac 中使用，可以使用 Ctrl + Alt 键释放。 选择安装过程中使用的语言↓ 选择 Linux 的安装位置↓ 设置 Linux 网络↓ 正式开始安装↓ 设置用户↓ 设置 root 账号密码↓ 添加新用户↓ 重启↓ 登录↓

接着，在菜单栏中选择 操作 → ⚙配置... （或者在 Linux 虚拟机右上角单击齿轮图标）

配置虚拟机的网络↓

默认就是 共享网络 。

共享网络是 Parallels Desktop 提供的一种网络连接方式，它使得 Linux 虚拟机会共享 Mac 当前使用的网络。

除此之外，还有桥接网络。此时虚拟机会被当作一台独立电脑连接网络。
